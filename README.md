# Uyiosa Nehikhuere — Portfolio

A personal portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step just something fun cuz im bored lol

## What's inside

- **WebGL gravity grid** background that reacts to your mouse and clicks
- **Smooth page transitions** with scroll, arrow keys, or the nav dots
- **Secret terminal** — there's a hidden command-line interface in here somewhere 👀
- **Bangers** font for the hero name, **Space Grotesk** for everything else

## File structure

```
portfilo/
├── index.html        # main entry point
├── css/
│   └── styles.css    # all styles
└── js/
    ├── cursor.js     # custom cursor
    ├── background.js # WebGL dot grid
    ├── pages.js      # page navigation
    ├── terminal.js   # secret terminal
    ├── welcome.js    # intro animation
    └── extras.js     # easter eggs & toast
```

## Running it

Just open `index.html` — no install or server needed.
